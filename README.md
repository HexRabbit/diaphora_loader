# diaphora_loader
A loader for IDA pro plugin diaphora

https://github.com/joxeankoret/diaphora

## Howto
Put your files like this
```
IDA/
└── plugins/
    ├── diaphora_loader.py
    └── diaphora/
        ├── ...
        └── diaphora.py
```
## Run
Enjoy diaphora from `File/Run diaphora` or `Edit/Plugins/diaphora`
